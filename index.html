@romatech/order-by

GitHub npm

Chainable multi-level sorting for JavaScript arrays via array.orderBy() and array.thenBy(). Immutable, fully typed, compatible with CommonJS and ESM.

Installation

npm install @romatech/order-by

Quick Start

import '@romatech/order-by';

const users = [
  { name: 'Alice', age: 30 },
  { name: 'Bob', age: 25 },
  { name: 'Charlie', age: 30 },
];

const sorted = users
  .orderBy(u => u.age)
  .thenBy(u => u.name);

console.log(sorted);
// [
//   { name: 'Bob', age: 25 },
//   { name: 'Alice', age: 30 },
//   { name: 'Charlie', age: 30 }
// ]

Features

Chainable Sorting

orderBy, orderByDesc, thenBy, thenByDesc — multi-level sorting made simple.

Immutable

Original array is never modified. Returns a new sorted array every time.

TypeScript Typed

Full typings with OrderedArray<T> interface and Selector<T> type.

Ascending & Descending

Sort in any direction with orderByDesc and thenByDesc.

Safe Prototype Extension

Non-enumerable properties — won't affect loops or JSON serialization.

Universal Module

Works with CommonJS (require) and ES Modules (import).
